# Avaliação

Jogo de Adivinhação

## Compilando

Clone o projeto, ou baixe a imagem ZIP do site.

Entre na pasta do projeto e digite:

    ./gradlew clean build
    
# Executando

Digite no console, a partir da pasta do projeto

    java -jar build/libs/gourmet-game.jar
